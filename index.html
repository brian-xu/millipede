Millipede

Millipede is a Flask app that generates websites from unsegmented text. It uses an LSTM architecture (as defined in this paper) and Jinja templates to quickly generate simple yet elegant static websites. In the past, artificially intelligent website builders have taken the route of creating a suitable shell for users to fill with text.

Millipede is an experiment in the other direction: given a body of text, divide it into distinct segments and create a website using them. This approach is more limited, as a result of limiting the user's control over the end result. However, this is also a unique strength of Millipede, as text bodies can quickly be converted to presentable websites with little effort from the user.

Millipede would be best suited for creating a readable landing page from informational text. An example is this website, which was generated using Millipede.

More complex websites such as a business's homepage are feasible with some slight tweaking to the resulting website. However, highly intricate and variable websites such as e-commerce platforms are better suited to other website builders.

Millipede is an open source project. The source repository is hosted on GitHub.
